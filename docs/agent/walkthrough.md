## 実施内容

- **評価基準の分岐**: `eval_spec.md` を修正し、ホテル紹介が主目的でないページ（ランニング等）では、直接の複数OTA設置ではなく「周辺ホテルへの動線とマイクロコピー」を評価するよう変更しました。
